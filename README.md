# Contractor Command Ops - Full Starter (Marketing Site + Portal + Dashboard)

This is a starter Next.js project with:
- Marketing pages (home, services, pricing, contact)
- Contact API endpoint
- Simple portal with login and a dashboard (token-based for demo)
- Styles and components

ENV:
- PORTAL_USER and PORTAL_PASS (for demo login) - set in Vercel dashboard
- AUTH_SECRET for signing tokens (change from default)
- Optional SMTP settings for emails (SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS, ADMIN_EMAIL)

Deploy:
1. Push to GitHub
2. Import to Vercel and set environment variables
3. Deploy
